# FL-GF

This is my own attempt on porting the old Guitar Flash game to kotlin. I am trying to do it by "cleaning" the old messy legacy project.

I don't expect this to be a full-featured product as original, btw it will be a good chance to see how games like Guitar Flash/Guitar Hero works under the hood.

# Dev stage

At this point, I've studied the old sources of **original** Guitar Flash game and, based on then, I extracted the idea of some models and started to consolidating an "engine" structure. I am also using Clean Architecture approach to help me separate concerns on this.
